概览

名称：Issue Englisher
目标：将中文/混合语的报错与描述，转为标准英文 GitHub Issue Markdown（含复现步骤、期望/实际、环境、建议标签）。
形态：单页 H5 + Serverless API
首发功能：单条生成、实时预览、复制、下载 .md、轻度限流、水印
可选付费：去水印、批量生成、团队风格包（本地/云）

功能规格（MVP）

表单与验证
必填：标题、问题描述、期望行为、实际行为
可选：复现步骤、环境信息、日志片段
校验规则：
字数下限：描述≥30字符
敏感信息遮挡：匹配 token/key/pass/secret/邮箱/IP，默认脱敏为 ****，并提示用户检查
占位符校验：不允许空的“1)”“2)”行；渲染时自动去空行

生成与模板
固定骨架 + 可选模块（Additional Context 若为空则省略）
语言：美式英文默认；可切英式拼写开关（付费）
术语风格：尽量使用直白工程英语（no fluff）
标签建议：基于文本启发式（bug/feature/docs/perf/build/ci）+ 模型判断

预览与导出
markdown-it + DOMPurify 安全渲染
下载文件名：issue-YYYYMMDD-HHMM.md
免费版页脚水印；付费移除

限流与收费
匿名：每日10次/IP;超出提示支付或次日重试
付费：解锁去水印 + 提额至每日200次 + 英式拼写切换
支付后发放 access_token（魔法链接携带，存本地）

提示词规范（Prompt）

系统原则（system）
角色：高级软件维护员，撰写清晰的GitHub问题。
目标：将多语言/非结构化输入重写为简洁、客观、可执行的英文 Issue。
风格：短句、无主观臆测、不给解决方案（只可在“Suspected cause”中以“if applicable”形式简述）。
结构：严格遵守模板；空字段不输出对应小节。

用户提示（user）
注入JSON（压缩后的字段），并附英式/美式拼写要求与标签建议开关。
要求：
标题：不超过 80 字符，包含受影响模块。
Steps to Reproduction：若用户未给，基于描述推断 3 步内最小复现;若仍无法，输出“Provide a minimal repro”占位。
Environment：按键值对整理;未知留空但不删节。
术语拼写：us/uk 一致化（color/colour， organize/organise）。
敏感数据不得回显;以 [REDACTED] 占位。

示例片段（assistant）
提供1-2条高质量 few-shot，覆盖常见前端/后端报错场景与日志脱敏示例。

输出约束（assistant）
只输出 Markdown 正文，不加解释。
末尾在免费模式下追加水印行。

限流与安全

速率限制：
Sliding window：IP+指纹 30 req/30min;匿名计数 10/day;token 用户 200/day
触发 429 时返回 Retry-After 秒数
内容安全：
正则遮挡：（？i）（api[-]？key|secret|token|password|bearer）[：=\\s]*（[A-Za-z0-9-]{8，}）
电子邮箱与IPv4/IPv6 脱敏
日志截断：单次提交日志上限 2,000 字符
XSS 防护：
后端只返回纯文本 Markdown
前端渲染前 DOMPurify.sanitize
滥用与刷量：
前端计数 + 后端二次校验
Cloudflare Turnstile/隐形验证码（可选）

// app/page.tsx
'use client'
import { useState } from 'react'
import DOMPurify from 'dompurify'
import Markdown from 'react-markdown'

export default function Home() {
  const [form, setForm] = useState({
    title: '', description: '', steps: [''],
    expected: '', actual: '',
    env: { os: '', appVersion: '', deps: '', logs: '' },
    options: { spelling: 'us', watermark: true, suggestLabels: true }
  })
  const [md, setMd] = useState(''); const [loading, setLoading] = useState(false)

  const submit = async () => {
    setLoading(true)
    const resp = await fetch('/api/generate', {
      method: 'POST', headers: { 'content-type': 'application/json' },
      body: JSON.stringify({
        ...form,
        steps: form.steps.filter(s => s.trim().length>0),
        env: { ...form.env, deps: form.env.deps.split(',').map(s=>s.trim()).filter(Boolean) },
        client: { ipHash: '', fingerprint: '' }
      })
    })
    const data = await resp.json()
    setMd(data.markdown || '')
    setLoading(false)
  }

  return (
    <div className="max-w-6xl mx-auto p-6 grid md:grid-cols-2 gap-6">
      <section>
        <h1 className="text-2xl font-bold">Issue Englisher</h1>
        {/* 表单控件省略，思路同理 */}

      </section>

    </div>
  )
}

// app/api/generate/route.ts
import { NextRequest, NextResponse } from 'next/server'
const OPENAI_API_KEY = process.env.OPENAI_API_KEY!

export async function POST(req: NextRequest) {
  const body = await req.json()
  // 1) 校验与脱敏（简化演示）
  if (!body.title || !body.description || !body.expected || !body.actual) {
    return NextResponse.json({ error: 'INVALID_INPUT' }, { status: 400 })
  }
  // 2) 组装提示
  const prompt = buildPrompt(body)
  // 3) 调用模型（伪代码，按所用SDK替换）
  const completion = await fetch('https://api.openai.com/v1/chat/completions', {
    method: 'POST',
    headers: { 'content-type': 'application/json', 'authorization': `Bearer ${OPENAI_API_KEY}` },
    body: JSON.stringify({
      model: 'gpt-4o-mini',
      messages: [
        { role: 'system', content: systemRules },
        { role: 'user', content: prompt }
      ],
      temperature: 0.2
    })
  }).then(r=>r.json())

  const markdown = completion.choices?.[0]?.message?.content || ''
  return NextResponse.json({
    markdown,
    labels: [], usage: { countToday: 1, limit: 10 }, cost: { tokens: 0, ms: 0 }
  })
}

const systemRules = `You are a senior maintainer... (略，按文档填写)`

function buildPrompt(input: any) {
  return `Please rewrite the following bug report into a clean GitHub Issue in US English.
Title: ${input.title}
Description: ${input.description}
Steps: ${(input.steps||[]).join('\n')}
Expected: ${input.expected}
Actual: ${input.actual}
Env: OS=${input.env?.os}; App=${input.env?.appVersion}; Deps=${(input.env?.deps||[]).join(', ')}
Logs: ${(input.env?.logs||'').slice(0,2000)}`
}
